# 소개
모주는 교육 목적의 모의 주식 투자 플랫폼으로, 학생들이 실제 자금 없이 안전한 환경에서 주식 투자를 체험하고 학습할 수 있도록 지원하는 서비스입니다.

"**모주**"라는 이름은 `모의 주식 투자 서비스`에서 파생되었습니다.
**모주**는 실제 주식 시장과 유사한 환경에서 투자 경험을 제공합니다.

간편한 수업 관리와 모니터링으로 관리자 경험을 향상시킬 수 있습니다. 또한 더 나은 투자 경험을 구축하는 데 도움이 되도록 실시간, 정확성 및 안정성의 모든 측면을 다루었습니다.

## 학생 서비스
### 수업 참여 & 대기 화면
- 학생이 수업 코드로 참여한 뒤, 수업 시작 전 상태를 확인하는 진입 화면입니다.
- 수업 시작 이벤트에 따라 다음 투자 흐름으로 자연스럽게 이어지도록 구성했습니다.

### 종목 조회 & 투자 판단 화면
- 학생이 종목 정보, 보유 현황, 투자 가능 자산을 확인하고 매수·매도 결정을 수행하는 핵심 화면입니다.
- 실제 수업 흐름에서 반복적으로 사용하는 화면인 만큼 정보 확인과 상호작용이 끊기지 않도록 구성했습니다.

### 뉴스 기반 투자 학습 화면
- 학생이 기사 내용을 바탕으로 시장 상황을 해석하고 투자 판단을 내릴 수 있도록 구성한 화면입니다.
- 단순 조회가 아니라 학습 맥락에서 투자 결정을 돕는 흐름에 집중했습니다.

### 결과 & 포트폴리오 화면
- 투자 결과와 팀별 성과를 확인하며 수업 흐름을 마무리하는 화면입니다.
- 수익률과 자산 변화를 한눈에 파악할 수 있도록 정리했습니다.

## 어드민 서비스
### 클래스 생성 & 수업 운영 화면
- 관리자가 클래스 생성, 차수 운영, 수업 상태 관리를 수행하는 핵심 운영 화면입니다.
- 실제 수업 진행 중 빠르게 상태를 확인하고 다음 액션으로 이어질 수 있도록 구성했습니다.
<img width="8176" height="2300" alt="image" src="https://github.com/user-attachments/assets/7c4e6615-f602-4b90-a978-0aaac1175c6b" />

### 종목 관리 & 수정 화면
- 관리자가 수업에 사용할 종목 정보를 등록·수정하고 반영 결과를 확인하는 화면입니다.
- 저장 이후 전체 화면을 다시 불러오지 않고 필요한 데이터만 갱신하도록 개선해 운영 흐름이 끊기지 않도록 했습니다.

### 기사 관리 화면
- 차수별 기사 등록과 수정, 노출 관리를 수행하는 관리자 화면입니다.
- 학생 투자 판단에 직접 연결되는 데이터인 만큼 관리 흐름을 단순하게 유지하는 데 집중했습니다.
