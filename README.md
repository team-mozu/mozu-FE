# 소개
모주는 교육 목적의 모의 주식 투자 플랫폼으로, 학생들이 실제 자금 없이 안전한 환경에서 주식 투자를 체험하고 학습할 수 있도록 지원하는 서비스입니다.

"**모주**"라는 이름은 `모의 주식 투자 서비스`에서 파생되었습니다.
**모주**는 실제 주식 시장과 유사한 환경에서 투자 경험을 제공합니다.

간편한 수업 관리와 모니터링으로 관리자 경험을 향상시킬 수 있습니다. 또한 더 나은 투자 경험을 구축하는 데 도움이 되도록 실시간, 정확성 및 안정성의 모든 측면을 다루었습니다.
- 실시간(Real-time) 투자 경험
- 투자 현황 모니터링
- 포트폴리오 관리 및 수익률 추적
- 수업(투자) 생성 및 관리
- 투자 종목 관리
- 차수 기사 관리
- 팀별 투자 현황 추적

**모주**는 단순한 모의 투자 도구를 넘어서, 미래 세대가 건전한 금융 의식과 투자 능력을 갖출 수 있도록 지원하는 종합적인 금융 교육 플랫폼입니다.
