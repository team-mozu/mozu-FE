# 모주(MOZU) - 프로젝트 도메인 개요

## 서비스 개요

**모주(MOZU)**는 교육 목적의 모의 주식 투자 플랫폼으로, 학생들이 실제 자금 없이 안전한 환경에서 주식 투자를 체험하고 학습할 수 있도록 지원하는 서비스입니다.

## 핵심 가치 제안

- **실무 중심 금융 교육**: 실제 주식 시장과 유사한 환경에서 투자 경험 제공
- **안전한 학습 환경**: 실제 자금 손실 없이 투자 전략 실험 가능
- **실시간 협력 학습**: 팀 단위 투자 활동을 통한 협업 능력 향상
- **교육자 지원**: 교사/강사가 수업을 효과적으로 관리할 수 있는 도구 제공

## 주요 사용자 그룹

### 1. 학생 (Student)
- **역할**: 모의 투자 참여자
- **주요 활동**: 
  - 수업 대기 및 참여
  - 팀 단위 주식 투자 결정
  - 포트폴리오 관리 및 수익률 추적
  - 실시간 투자 현황 모니터링

### 2. 교육자/관리자 (Admin)
- **역할**: 수업 운영자 및 관리자
- **주요 활동**:
  - 수업(Class/Lesson) 생성 및 관리
  - 투자 종목(Stock) 관리
  - 학습 자료(Article) 관리
  - 실시간 수업 모니터링
  - 팀별 투자 현황 추적

## 핵심 도메인 모델

### 1. 수업 (Class/Lesson)
- **정의**: 특정 기간 동안 진행되는 모의 투자 교육 세션
- **구성 요소**:
  - 수업명, 기간, 참여 팀 수
  - 초기 투자 자금 설정
  - 투자 라운드 수 설정
  - 투자 가능 종목 목록
  - 각 라운드별 학습 자료

### 2. 투자 종목 (Stock/Item)
- **정의**: 학생들이 투자할 수 있는 가상의 주식 종목
- **구성 요소**:
  - 종목명, 종목 정보, 로고
  - 재무 정보 (자금, 부채, 자본, 수익성 지표)
  - 라운드별 가격 변동 데이터

### 3. 학습 자료 (Article)
- **정의**: 투자 판단에 도움을 주는 뉴스, 분석 자료
- **구성 요소**:
  - 제목, 내용, 이미지
  - 공개 시점 (특정 투자 라운드)
  - 관련 종목 정보

### 4. 팀 (Team)
- **정의**: 함께 투자 결정을 내리는 학생 그룹
- **구성 요소**:
  - 팀명, 팀원 목록
  - 포트폴리오 (보유 종목, 현금)
  - 투자 히스토리
  - 수익률 및 순위

## 서비스 플로우

### 1. 수업 준비 단계
1. **관리자**: 새로운 수업 생성
2. **관리자**: 투자 가능 종목 선정 및 설정
3. **관리자**: 라운드별 학습 자료 준비
4. **관리자**: 수업 시작 및 참여 코드 공유

### 2. 수업 진행 단계
1. **학생**: 참여 코드로 수업 입장
2. **학생**: 팀 구성 및 대기
3. **각 투자 라운드**:
   - 학습 자료 공개
   - 팀별 투자 결정 및 거래
   - 결과 확인 및 포트폴리오 업데이트
4. **관리자**: 실시간 모니터링 및 다음 라운드 진행

### 3. 수업 마무리 단계
1. **시스템**: 최종 수익률 계산
2. **시스템**: 팀별 순위 산정
3. **관리자**: 결과 리뷰 및 피드백
4. **학생**: 학습 성과 확인

## 핵심 기능 영역

### 1. 실시간 동기화 (Real-time Synchronization)
- **기술**: Server-Sent Events (SSE)
- **목적**: 
  - 수업 진행 상황 실시간 동기화
  - 투자 라운드 시작/종료 알림
  - 팀별 투자 현황 업데이트

### 2. 협업 투자 (Collaborative Trading)
- **특징**: 팀 단위 의사결정
- **기능**:
  - 팀원 간 투자 의견 공유
  - 합의 기반 투자 결정
  - 팀 포트폴리오 관리

### 3. 학습 지원 (Educational Support)
- **자료 제공**: 라운드별 맞춤 학습 자료
- **분석 도구**: 수익률, 위험도 등 투자 성과 분석
- **피드백**: 투자 결과에 대한 교육적 해석

### 4. 관리 도구 (Management Tools)
- **수업 관리**: 생성, 수정, 삭제, 모니터링
- **콘텐츠 관리**: 종목, 학습 자료 관리
- **진행 관리**: 라운드 진행, 타이밍 조절

## 교육적 목표

### 1. 금융 이해력 향상
- 주식 시장 메커니즘 이해
- 기업 재무 정보 분석 능력
- 리스크와 수익의 관계 파악

### 2. 의사결정 능력 개발
- 정보 기반 투자 판단
- 불확실성 하에서의 선택
- 결과에 대한 책임감

### 3. 협업 능력 강화
- 팀 내 의견 조율
- 역할 분담 및 책임 공유
- 집단 지성 활용

### 4. 디지털 금융 리터러시
- 온라인 거래 플랫폼 사용법
- 디지털 정보 활용 능력
- 기술 기반 금융 서비스 이해

### 1. 다양한 금융 상품
- 채권, 펀드, 파생상품 등으로 확장

- 성과 분석 대시보드

### 3. 소셜 러닝 기능
- 투자 전략 공유
- 커뮤니티 기반 학습
- 멘토링 시스템

### 4. 게임화 요소
- 성취 시스템
- 리더보드
